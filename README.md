## DADI

### RACCOLTA DATI
- generare un numero random da 1 a 6, sia per il giocatore sia per il computer

### ESECUZIONE LOGICA
- stabilire il vincitore in base a chi fa il punteggio più alto

### OUTPUT
- stampare il punteggio del vincitore
